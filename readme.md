Getting Started
---------------
The easiest way to get started is to clone the repo:

```bash
# Get the latest version
git clone --depth=1 https://github.com/gkiely/Batch.git batch

# Change directory
cd batch

# Install NPM dependencies
npm i

# Start
npm start
```
